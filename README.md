# Fanuc_Annotation_tool
发那科工业机器人注释工具

# 文件介绍：
1、main2.py为主程序入口
2、其他文件为UI文件或多线程文件
3、img文件夹储存着程序所需的图片资源，img.qrc为Pyqt引用图片的方法

# 项目介绍：
本项目由不知名网友开发及维护，微信公众号：不知名网友i，与2023/8/7停止维护和更新，请不要将此代码用于商业用途。
